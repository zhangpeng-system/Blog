## 平仮名(ひらがな)

平假名是日语中表音文字的一种。平假名是从中文汉字的草书演化而来的。

![](https://peridot-wood-05b.notion.site/image/https%3A%2F%2Fprod-files-secure.s3.us-west-2.amazonaws.com%2F9eac8f3d-2b0a-48f1-890e-bf2567cf11ae%2Fc6421943-0e7d-4251-bbd4-d7e80aa154ea%2FUntitled.png?table=block&id=a2201f16-39d9-4785-b3b0-b7e644b94ab6&spaceId=9eac8f3d-2b0a-48f1-890e-bf2567cf11ae&width=2000&userId=&cache=v2)

## カタカナ(片假名)

片假名，是日语中表音符号（音节文字）的一种。片假名是从中文汉字的楷书或部首演化而来的。

![](https://peridot-wood-05b.notion.site/image/https%3A%2F%2Fprod-files-secure.s3.us-west-2.amazonaws.com%2F9eac8f3d-2b0a-48f1-890e-bf2567cf11ae%2F574c1abf-0f39-40a0-879b-defb32d89671%2FUntitled.png?table=block&id=d9fe025f-2f14-4056-bc12-a21536b8269b&spaceId=9eac8f3d-2b0a-48f1-890e-bf2567cf11ae&width=2000&userId=&cache=v2)

## 五十音

五十音（ごじゅうおん）または五十音図（ごじゅうおんず）とは、日本語の仮名文字を母音に基づき縦に五字、子音に基づき横に十字ずつ並べたもの。

### 清音（せいおん）

母音（ぼおん）：あ、い、う、え、お。

子音（しおん）：あ、か、さ、た、な、は、ま、や、ら、わ。

|                | あ、ア段（a） | い、イ段(i)  | う、ウ段(u)     | え、エ段(e)   | お、オ段(o) |
| -------------- | ------------- | ------------ | --------------- | ------------- | ----------- |
| あ、ア行(a)    | あ、ア(a)     | い、イ(i)    | う、ウ(u)       | え、エ(e)     | お、オ(o)   |
| か、カ行(k)    | か、カ(ka)    | き、キ(ki)   | く、ク(ku)      | け、ケ(ke)    | こ、コ(ko)  |
| さ、サ行(s)    | さ、サ(sa)    | し、シ(si)   | す、ス(su)      | せ、セ(se)    | そ、ソ(so)  |
| た、タ行(t,ts) | た、タ(ta)    | ち、チ(chi)* | つ、ツ(tsu,tu)* | て、テ(te)    | と、ト(to)  |
| な、ナ行(n)    | な、ナ(na)    | に、ニ(ni)   | ぬ、ヌ(nu)      | ね、ネ(ne)    | の、ノ(no)  |
| は、ハ行(h)    | は、ハ(ha)    | ひ、ヒ(hi)   | ふ、フ(hu,fu)*  | へ、ヘ(he,e)* | ほ、ホ(ho)  |
| ま、マ行(m)    | ま、マ(ma)    | み、ミ(mi)   | む、ム(mu)      | め、メ(me)    | も、モ(mo)  |
| や、ヤ行(y)    | や、ヤ(ya)    |              | ゆ、ユ(yu)      |               | よ、ヨ(yo)  |
| ら、ラ行(r)    | ら、ラ(ra)    | り、リ(ri)   | る、ル(ru)      | れ、レ(re)    | ろ、ロ(ro)  |
| わ、ワ行(w)    | わ、ワ(wa)    |              |                 |               |             |

### 拗音（ようおん）

变化规则：在い段和え段的后面增加や行的子音，拼音之后即可得出拗音的正确读法。

| や（ya）  | ゆ(yu)    | よ(yo)    |
| --------- | --------- | --------- |
| きゃ(kya) | きゅ(kyu) | きょ(kyo) |
| しゃ(sya) | しゅ(syu) | しょ(syo) |
| ちゃ(cya) | ちゅ(cyu) | ちょ(cyo) |
| にゃ(nya) | にゅ(nyu) | にょ(nyo) |
| ひゃ(hya) | しゅ(hyu) | ひょ(hoo) |
| みゃ(mya) | みゅ(myu) | みょ(myo) |
| りゃ(rya) | りゅ(ryu) | りょ(ryo) |

### 濁音（だくおん）

变化规则：多两点。

|             | あ段     | い段     | う段    | え段   | お段   |
| ----------- | -------- | -------- | ------- | ------ | ------ |
| が、ガ行(g) | が（ga） | ぎ（gi） | ぐ(gu)  | げ(ge) | ご(go) |
| ざ行(z)     | ざ(za)   | じ(ji)*  | ず(zu)  | ぜ(ze) | ぞ(zo) |
| だ行(d)     | だ(da)   |          | づ(du)* | で(de) | ど(do) |
| ば行(b)     | ば(ba)   | び(bi)   | ぶ(bu)  | べ(be) | ぼ(bo) |

### 半濁音（はんだくおん）

变化规则：多一个小圆圈。

|         | あ段   | い段   | う段   | え段   | お段   |
| ------- | ------ | ------ | ------ | ------ | ------ |
| ぱ行(p) | ぱ(pa) | ぴ(pi) | ぷ(pu) | ぺ(pe) | ぽ(po) |

### 長音（ちょうおん）

变化规则：平假名和片假名的变化如下

|平仮名| 正常 | 長音 |
| ---- | ---- | ---- |
| あ   | あ   | ああ |
| い   | い   | いい |
| う   | う   | うう |
| え   | え   | ええ |
| お   | こ   | おお |

|カタカナ| 正常 | 長音 |
| ---- | ---- | ---- |
| ア   | ア   | アー |
| イ   | イ   | イー |
| ウ   | ウ   | ウー |
| エ   | エ   | エー |
| オ   | オ   | オー |

### 促音（そくおん）

仮名ではつ、ツで書き表され、普通の「つ」と区別するため、一般に「っ」、「ッ」のように小さく書かれる。

例：清楚的，はっきり、ハッキリ、hakkiri。

### 撥音（はつおん）拨音

仮名ではつ、ツで書き表され、普通の「つ」と区別するため、一般に「っ」、「ッ」のように小さく書かれる。

例：音、おん、オン、onn。
