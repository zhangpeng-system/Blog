# なぜ私は最終的にChatGPTとGitHub Copilotだけを残したのか

2023年頃から、AIという言葉を聞かない日はなくなった。
当時はまだ日本語の情報も少なく、何ができて何ができないのかもよく分からなかった。

それから数年。

気が付けば私は様々なAIを試していた。

ChatGPT。Claude。
Gemini。

GitHub Copilot。

Cursor。

Claude Code。

Codex。

その他にも名前を忘れてしまったサービスがいくつもある。

新しいモデルが出れば試す。

新しいAI IDEが出れば試す。

Agentが流行れば調べる。

MCPという言葉を見れば調べる。

今振り返ると、AIを使う時間よりもAIについて調べる時間の方が長かったかもしれない。

---

# AIを追いかける事が目的になっていた

ある時ふと思った。

私は何のためにAIを使っているのだろう。

新しいモデルが発表される。

SNSでは性能比較が始まる。

ベンチマーク。

コンテキスト長。

推論能力。

Agent機能。

ツール呼び出し。

気が付けば毎日のように比較記事を読んでいた。

しかし翌日会社へ行くと現実は変わらない。

私が向き合っているのは、

最新のAI研究ではなく、

OCRシステムだった。

Oracleだった。

Stored Procedureだった。

VB.NETだった。

何年も動き続けている業務システムだった。

AIの進化は確かに面白い。

しかし私の日常は、そこまで劇的には変わらない。

そんな当たり前の事に少しずつ気付き始めた。

---

# 本当に欲しかったのは「最強のAI」ではなかった

AIを比較し続けていた頃、

私は無意識に「最強のAI」を探していた。

しかしある時から考え方が変わった。

本当に必要だったのは、

最強のAIではなく、

自分の仕事や生活に自然に溶け込むAIだった。

例えば私は普段VSCodeを使う。

その中でコードを書き、

調査をし、

修正を行う。

そこに自然に存在してくれる方が重要だった。

また、家に帰ればブログを書いたり、勉強したり、将来について考えたりする。

そういう時に長時間対話できる相手の方が重要だった。

結局のところ、

AIそのものよりも、

自分の生活との相性の方が大切だった。

---

# ChatGPTは思考空間になった

今ではChatGPTを毎日のように使っている。

しかしコードを書くためではない。

もちろんコードについて相談する事もある。

だが実際にはそれ以外の方が多い。

帰化申請の準備。

住宅購入の検討。

ブログ記事。

ブランド構想。

キャリア。

学習計画。

将来の不安。

日常の悩み。

様々な事を話している。

気が付けばChatGPTは検索エンジンの代わりではなくなっていた。

私にとっては、

考えを整理する場所になっていた。

何かを決める時。

何かに迷った時。

新しい事を学ぶ時。

頭の中にある曖昧な考えを言葉にする時。

そういう場面で最も役に立っている。

---

# GitHub Copilotは疲労を減らしてくれた

一方で開発中に最も使うAIはGitHub Copilotになった。

最初は懐疑的だった。

コード補完なんて大した事はないだろうと思っていた。

しかし使い続けると印象が変わった。

Copilotは仕事を代わりにやってくれる訳ではない。

むしろ、

疲労を減らしてくれる。

という表現の方が近い。

SQLを書く。

DTOを作る。

同じようなコードを繰り返し修正する。

メソッドを追加する。

こうした作業は難しくない。

しかし集中力を消耗する。

Copilotはその消耗を減らしてくれる。

そのおかげで、

本当に考えるべき部分に時間を使えるようになった。

私にとってCopilotは、

プログラマーを置き換える存在ではなく、

隣に座っている補助スタッフのような存在になっている。

---

# ClaudeやAgentに感じる未来

もちろんChatGPTとCopilot以外に価値がない訳ではない。

Claudeは長文の理解能力が高いと感じる。

大量の資料を読む時には非常に便利だ。

Cursorも面白い。

プロジェクト全体を理解して修正する能力には驚かされた。

Claude CodeやAgent系のツールも興味深い。

初めて触った時は少し未来を感じた。

ただ、今の私の仕事環境ではまだ研究対象という位置付けに近い。

老朽化した業務システム。

長年積み重なった業務ロジック。

限られたドキュメント。

そうした現場では、

まだ人間がハンドルを握る必要があると感じている。

AIに任せる部分。

自分で確認する部分。

その境界線を見極める事が大切なのだと思う。

---

# AIを研究していて分かった事

AIについて調べ続けた数年間で分かった事がある。

それは、

AIよりも自分自身の方が重要だという事だ。

どんな道具を使うか。

どんな働き方をしたいのか。

どんな生活を送りたいのか。

どんな人間になりたいのか。

そうした事はAIが決めてくれない。

AIは便利な道具だ。

しかし人生の方向を決めるコンパスではない。

だから私は最近、

以前ほど新しいAIを追いかけなくなった。

もちろん興味はある。

新しい技術も好きだ。

しかし、それ以上に大切なのは、

今の生活の中でどう活用するかだと思っている。

---

# 最後に

数年前の私は、

もっと強いAIを探していた。

もっと賢いモデルを探していた。

もっと便利なツールを探していた。

しかし今の私は少し違う。

最終的に残ったのは、

ChatGPTとGitHub Copilotだった。

決して最強だからではない。

私の生活や仕事の中で自然に使い続けられたからだ。

AIの世界はこれからも変化し続けるだろう。

数年後には今とは全く違う環境になっているかもしれない。

それでも一つだけ変わらないと思う。

どれだけ優秀なAIが登場しても、

最後に何を作るのか。

どこへ向かうのか。

それを決めるのは人間自身なのだと思う。
